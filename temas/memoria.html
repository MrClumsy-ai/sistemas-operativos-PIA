Almacenaje, memoria y archivos

En Android, todo lo que tiene que ver con almacenamiento, memoria y archivos está diseñado para que el sistema funcione de manera eficiente y segura, sin memoria este no tiene en donde almacenar la informacion para su funcionamiento correcto, por eso todo dispositivo requiere de una memoria no importa que tan pequeña sea ya que sin esta dificilmente funcionaria

Almacenamiento

El almacenamiento se divide en dos partes: el interno y el externo. El interno es como un espacio privado donde cada aplicación guarda su información sin que otras puedan meterse, perfecto para configuraciones y datos importantes. El almacenamiento externo es más accesible, ideal para fotos, videos y documentos, pero en versiones recientes de Android hay más restricciones para evitar problemas de seguridad. Estas suelen ser las memorias SSD. Diciendolo mas general es el espacio donde se guardan de manera permanente las aplicaciones, los archivos personales como fotos, videos, documentos y también el propio sistema operativo. Este almacenamiento puede ser interno, que es el que viene dentro del dispositivo, o externo, como una tarjeta de memoria que se puede agregar para aumentar la capacidad de guardado.

Memoria RAM

La memoria RAM es lo que permite que las aplicaciones funcionen sin trabas. Android es bastante inteligente en esto: si muchas apps están abiertas y la memoria se llena, el sistema cierra las menos importantes para que todo siga yendo rápido. También usa caché, una especie de memoria temporal que ayuda a que las aplicaciones se abran más rápido la próxima vez que las uses.Esta memoria permite que el teléfono trabaje de manera más rápida y pueda mantener varias tareas abiertas al mismo tiempo. Cuando el dispositivo se apaga, todo lo que estaba en la memoria RAM se borra, la ram no es expclusivamente de los dispositivos moviles ya que las computadoras tambien hacen uso de estas, y entre mas ram tenga un dispositivo mas procesos podra llevar acabo haciendo que sea mas rapida cumpliendo su funcion

Archivos

Los archivos en Android están organizados en un sistema llamado ext4, que básicamente mantiene todo en orden. Cada aplicación tiene su propio espacio separado para guardar sus cosas sin interferencias, para acceder a los archivos, Android ofrece diferentes herramientas según el tipo de dato: por ejemplo, hay una para fotos y videos, otra para documentos y otra para archivos internos de las apps. En pocas palabras, Android se encarga de administrar el almacenamiento y la memoria de manera que todo fluya bien, sin desperdiciar recursos y manteniendo la seguridad de tus datos.Incluyen el núcleo o kernel, que conecta el hardware con el software, los servicios básicos que permiten operar las aplicaciones, y las aplicaciones propias del sistema como las de llamadas, mensajes, configuración, entre otras. Estos archivos son protegidos para evitar modificaciones accidentales, ya que son necesarios para que el dispositivo funcione correctamente.
